# project-management-sample-frontend

[project-management-sample](https://github.com/akinami3/project-management-sample) のフロントエンドリポジトリです。

## 技術スタック

| カテゴリ | 技術 |
|---|---|
| フレームワーク | React |
| 言語 | TypeScript |
| Lint | ESLint, Prettier |

## セットアップ

```bash
git clone https://github.com/akinami3/project-management-sample-frontend.git
cd project-management-sample-frontend
npm install
npm run dev
```
